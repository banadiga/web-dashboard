Web dashboard
=============

How to use
----------

* Create own `src/api/dashboard.json`. To more details go to [How to create dashboard.json](/src/api)
* Run application on server

How to build
------------

Go to `src/` and run

```bash
bower install
```

How to run locally
------------------

To run locally use `server`
To more details go to [How to use server](/ops)
